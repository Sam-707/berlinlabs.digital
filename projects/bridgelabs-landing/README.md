## About

BridgeLabs is a Berlin-based digital solutions agency specializing in:

- **Web Development** - Modern, fast websites built with React 19, TypeScript, and Vite
- **SaaS Platforms** - Full-cycle subscription products with multi-tenant architecture
- **Real-time Features** - Live updates using Supabase Realtime
- **Mobile-First Design** - Responsive interfaces optimized for all devices
