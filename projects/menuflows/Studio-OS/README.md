# Studio-OS Folder

This folder contains the Studio OS bone. Copy or rename it per project.

Quick start: see `FIRST_PROJECT_CHECKLIST.md`.

## Use in a new project

1. Copy this folder into the new repo root.
2. Rename the folder to the project name if desired.
3. Move contents to repo root if you want the structure at top-level.
4. Run `./scripts/studio.sh` after making scripts executable.
5. Keep the docs updated as you ship.

## Notes

- If you move contents to repo root, keep `AI_RULES.md`, `CLAUDE.md`, and `RUNBOOK.md` at the top level.
- If you keep this as a folder, point tools to `Studio-OS/` paths.
