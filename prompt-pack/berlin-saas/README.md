# Berlin Solopreneur SaaS Prompt Pack

> **A reusable, modular prompt system for rapidly building SaaS/web applications as a solopreneur**

---

## Overview

This prompt pack is designed for Berlin-based solopreneurs building production web applications with modern technologies. It provides modular, copy-paste prompts for every stage of SaaS development.

**Tech Stack:**
- React 19 + TypeScript + Vite
- Supabase (PostgreSQL + Auth + Real-time)
- Custom CSS (no Tailwind)
- Row Level Security (RLS) enforced

## Pro Tips

1. Always specify your tech stack upfront
2. Start small - build core features first
3. Use troubleshooting prompts before deploying
4. Customize RLS rules based on security needs
5. Keep consistent UI patterns across projects
6. Version control your prompts
7. Cache AI responses to save costs
8. Use Edge Functions for API key logic
9. Test WhatsApp bots with Twilio sandbox first
